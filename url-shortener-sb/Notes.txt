"Auto-configuration" feature in Spring Boot. 
As we have added the dependency to connect with MySQL in pom.xml. Spring Boot will see the dependency and will auto-configure MySQL in the project i.e. it will create connection with the data source. What we have to do is provide application/project the credentials to connect to the database in application.properties file.

Below tells how scema should be managed in database. This will update the schema automatically if we update the models
spring.jpa.hibernate.ddl-auto=update

UserDetailsImpl class is the class that will bridge the gap between custom users model created by us and Spring Security's user model.
Spring security needs to know against which user is it authenticating. UserDetails that has been implemented by UserDetailsImpl class is the Spring Security class.
